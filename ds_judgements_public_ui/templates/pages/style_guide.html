Find case law UI style guide

This page is a repository of UI elements used on the site. Developers can copy the code to add these components to FCL projects.

Components

Checkboxes

They can contain multiple items which have an associated ‘further details’ checkbox.

What type of organistation is it?
Select all that apply.

Radios

They may contain error messages also can progressively reveal subfields (eg ‘details of the person responsible for compliance’)

Who is the person in your organistion who will be responsible for licence compliance?
Select the correct option below

Error: Email address cannot be blank

Text

Input should have an optional heading and can include “hint text, error state, plus an error message

Enter any other names your organisation is known by

Error: If your organisation is known by any other names, please type name

Character count

Help users know how much text they can enter when there is a limit on the number of characters.

a. the risks you have identified, and
b. how you will protect the ability of the courts and tribunals to function

You should consider anything that may undermine the ability for the justice system to operate impartially and ...

Maximum 100 words

You can enter up to 100 characters

Summary card

If you’re showing multiple summary lists on a page, you can show each list within a summary card. This lets you visually separate each summary list and give each a title and some actions.
