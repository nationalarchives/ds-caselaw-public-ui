# Card Jinja Macros

Use the card component to help users navigate through your service. For example, this could be used on:
- homepages
- index pages
- hub pages
This page shows the Jinja macros for the Card component alongside rendered examples.

---

## Wrapper Macro

<Source
  language="jinja"
  code={`{% macro default_card() %}
<h2>Card</h2>
<div class="components__grid">
      {{ card_examples() }}
</div>
{% endmacro %}`}
  dark
/>

<Story name="AllCards" of={AllCards} />

## Default Card Macro

<Source
  language="jinja"
  code={`{% macro default() %}
  {% call card(title="Default") %}
    <p>This is the default card style</p>
  {% endcall %}
  {% call card(title="Default - with link", url=url("components")) %}
    <p>This is a default card with a link</p>
  {% endcall %}
{% endmacro %}`}
  dark
/>

<Story name="DefaultCard" of={DefaultCard} />

## Light Card Macro

<Source
  language="jinja"
  code={`{% macro light() %}
{% call card(title="Light", variant="light") %}
    <p>This is the light card style</p>
  {% endcall %}
  {% call card(variant="light") %}
    <button class="button-primary">Button</button>
    <p>Just a light example with a button</p>
  {% endcall %}
{% endmacro %}`}
  dark
/>

<Story name="LightCard" of={LightCard} />

## Image Card Macro

<Source
  language="jinja"

  {% endcall %}
