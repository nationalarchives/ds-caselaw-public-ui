# Alert Jinja Macros

Use an alert to tell the user about something they need to know about directly related to the page content.
This page shows the Jinja macros for the Alert component alongside rendered examples.

---

## Wrapper Macro

<Source
  language="jinja"
  code={`{% macro default_alert() %}
<h2>Alert</h2>
<div class="components__grid">
      {{ alert_examples() }}
</div>
{% endmacro %}`}
  dark
/>

<Story name="AllAlerts" of={AllAlerts} />

---

## Default Alert Macro

<Source
  language="jinja"
  code={`{% macro default() %}
  {% call alert(title="Title") %}
    <p>Content</p>
  {% endcall %}
{% endmacro %}`}
  dark
/>

<Story name="DefaultAlert" of={DefaultAlert} />
